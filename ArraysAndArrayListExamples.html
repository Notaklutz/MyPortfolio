Counting Cards

The CountingCards class will read a set of four strings, representing four hands in the game of bridge, from a file called bridgeHands.txt. It will sort the cards and output them in descending order by suit. It uses multiple ArrayLists to accomplish this task. Press the buttons below to scroll through the images of the CountingCard class. Below those images is the CountingCardApp class, which is the driver class for the program.
