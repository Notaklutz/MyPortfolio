Arrays and ArrayList

 

Arrays

Arrays hold blocks of memory, which hold values of the same data type. A value in the array is called an element and the position that it is at is called an index. The whole array is referred to by one name and individual elements in the array can be accessed using the array's indices. The index starts at 0 and ends at -1 the array's length. If an attempt to access an element at an invalid index is made, an ArrayIndexOutOfBoundsException is thrown. Furthermore, an array is declared with a fixed length that cannot be changed. Arrays are advantageous because they are one object that holds many values and each element can be directly accessed using its index. An important thing to note about arrays is that when an array is passed into a method, it is considered a pass by reference. This means that any changes made to the array in the method will occur to the original array. The elements of the array are not copied like with other data types.

An array can store multiple values of the same data type. (Photo: The Programmer Toolbox)

 

Two-Dimensional Arrays

Two-dimensional (2-D) arrays are arrays with columns and rows. They can be used to represent board games, tables, mazes, etc. Each row or column in a 2-D array is allowed to have a different number of elements (a ragged array). For example, the first row of the array could have 5 elements, but the second row of the array could have 3.

ArrayList

An ArrayList (java.util.ArrayList) is an implementation of a list using an array. It is very similar to an array and can hold objects of any data type except primitive data types (int, boolean, etc.) unless they are in a wrapper class (Integer, Boolean, etc.). is the type parameter for a generic object, but it can be changed to store objects of any class. Whenever the ArrayList runs out of space, its capacity automatically doubles (a bigger array is created and the values from the original array are copied). Elements in an ArrayList cannot be accessed directly. Instead, ArrayList.get(int index) and ArrayList.set(int index, E element) are used to get and set the elements in an ArrayList. Other methods such as add() can be used to add elements to the ArrayList.
